Overview of IR Projects

Retrieval Models, Indexing, Machine Learning

  • Developed a Python code and algorithm to efficiently index a large collection of text documents in hard disk and also to index in Elasticsearch using API.
  • Calculated scores based on Vector space (Okapi-25, TF-IDF,BM-25) and language models (Jelinek Mercer and Laplace Smoothing) for given queries and obtained good precision values.
  • Implemented Decision Tree algorithm from Python Sci-Kit library to train and test data by using the calculated scores as features, the obtained results showed good precision on test data.

Web Crawler, Vertical Search and Page Rank

  • Developed a Python code to crawl around 30,000 URLs and combined crawls from fellowmates using distributed indexing feature of ElasticSearch to form web graph.
  • Developed a python script to calculate F1 measure, precision, Mean-precision.
  • Implemented the page rank algorithm on the graph.
